# pochipochi

家計簿アプリ

## 開発環境セットアップ

### 前提条件

- [mise](https://mise.jdx.dev/) がインストールされていること

### セットアップ手順

1. リポジトリをクローン

```bash
git clone <repository-url>
cd pochipochi
```

2. mise でNode.js をインストール

```bash
mise install
```

3. Node.js のバージョンを確認

```bash
node --version
```

### 使用技術

- Node.js 22.16.0 (mise で管理)
